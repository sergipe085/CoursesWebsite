## 🚀 How to run the project

#### Clone the repository
```bash
git clone https://github.com/<GITHUB_USER>/<REPO_NAME>.git
```

#### Install requirements
```bash
cd CoursesWebsite
pip install -r requirements.txt
```

#### Run the application
```bash
python -m flask run
```
